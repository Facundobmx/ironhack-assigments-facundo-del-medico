Verdadero o Falso: CSS es un lenguaje de programación. ¿Por qué?

Segun lo que he investigado, tecnicamente seria falso, por tanto no es un lenguaje de programacion propiamente dicho. De todas maneras si es un leguanje que tiene una sintaxis que esta relacionado con lenguajes de programacion, como Javascript como ejemplo, colaborando asi de manera directa para generar resultados esteticos en una pagina web.

¿Para qué se utiliza CSS?

CSS se utiliza como una hoja de estilo en conjunto con HTML y JAVASCRIPT, basicamente atravez de la sintaxis de CSS podemos modificar las propiedades del HTML por ejemplo, Diy obtener resultados a nivel esteticos deseados

¿Cómo puedes agregar estilos a tu proyecto HTML?

Para agregar estilos CSS en mi HTML lo correcto seria separar ambos archivos. Supongamos que tenemos un INDEX.HTML, lo correcto seria crear un archivo CSS para ese INDEX.HTML (Por ejemplo INDEX.CSS o STYLE.CSS). Una vez creado el archivo del que hablamos sera necesario hacer un link del INDEX.CSS en el INDEX.HTML. Para ello deberemos usar la etiqueta "a href="../../INDEX.CSS" dentro del "head", una vez logrado esto dentro del INDEX.CSS comenzaremos a dar estilo a nuestro INDEX.HTML obviamente siguiendo la sintaxis de CSS.
